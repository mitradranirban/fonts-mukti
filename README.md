MUKTI OPEN SOURCE OPEN TYPE BANGLA FONT
=======================================

This is a one of the earliest Open Source OpenType Bengali / Bangla font. It was first made in 2002 using good quality glyphs of GPLed font bng2-n from [Cyberscape Multimedia](https://web.archive.org/web/20021113130716/http://www.akruti.com/freedom/). It was made for [Mukta Bangla Font project](https://nongnu.org/freebangfont).

This font is a part of debian package [fonts-beng-extra](https://packages.debian.org/source/sid/fonts-beng-extra)
and is available in Fedora distribution as [fbf-mukti-fonts](https://packages.fedoraproject.org/pkgs/fbf-mukti-fonts/fbf-mukti-fonts/index.html)

You can download the recent version 3.4.3 from my [fonts site](https://fonts.atipra.in/mukti.html)
